# NexaPanel — Install & Repair Scripts

  Official installation and repair scripts for **NexaPanel** Linux server control panel.

  ## Quick Install (Fresh VPS)

  ```bash
  bash <(curl -fsSL https://nexapanel.hostganga.com/install.sh || curl -fsSL https://raw.githubusercontent.com/payoro11/nexapanel-install/main/install.sh)
  ```

  ## Smart Repair (Fix Any Issue)

  ```bash
  bash <(curl -fsSL https://nexapanel.hostganga.com/repair.sh || curl -fsSL https://raw.githubusercontent.com/payoro11/nexapanel-install/main/repair.sh)
  ```

  ## Diagnose Only (No Changes)

  ```bash
  bash <(curl -fsSL https://nexapanel.hostganga.com/repair.sh || curl -fsSL https://raw.githubusercontent.com/payoro11/nexapanel-install/main/repair.sh) --check-only
  ```
